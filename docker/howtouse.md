# EKF-LOAM

Original repository: https://github.com/ITVRoC/ekf_loam


## Build
```bash
cd ekf_loam/docker
sudo ./build.sh
```


### Open docker image:
```bash
cd ekf_loam/docker
sudo ./run.sh

roslaunch ekf_loam ekf_loam.launch
```
